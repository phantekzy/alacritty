#  Phantekzy Alacritty Config (Windows)

A professional-grade Alacritty configuration optimized for **Windows 11** and **Neovim**. This setup bridges the gap between Windows and the "Kitty" Linux terminal experience.

##  Pro Features
* **GPU Accelerated:** Zero-latency text rendering for deep coding sessions.
* **Kitty Parity:** Custom keybindings for `F11 Fullscreen`, `Ctrl+Shift+C/V` clipboard, and dynamic font scaling.
* **Unified Aesthetics:** Full **Tokyonight (Night)** color palette integration.
* **Dev Workflow:** Defaults to PowerShell with an automated entry path to the Desktop.
* **Typography:** Precision-tuned for **FiraCode Nerd Font** with built-in box drawing for Neovim UI.

##  Installation
1. Install [Alacritty](https://github.com/alacritty/alacritty/releases).
2. Move `alacritty.toml` to `%APPDATA%/alacritty/`.
3. Restart Alacritty and hit `F11` for the full experience.
